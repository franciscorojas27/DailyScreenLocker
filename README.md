# RapiGo

RapiGo — Proyecto Go ligero y veloz

Descripción
- RapiGo es un proyecto en Go pensado como plantilla ligera para utilities rápidas y eficientes.

Características
- Estructura mínima con `main.go` y `go.mod`.
- Licencia MIT.

Instalación
```bash
go build -o rapigo
```

Uso
```bash
./rapigo
```
